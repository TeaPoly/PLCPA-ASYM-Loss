# PLCPA loss

The power-law compressed phaseaware (PLCPA) loss

The definition is as follows:

$$
\mathcal{L}_{PLCPA}=\left||S|^p-|\hat{S}|^p\right|^2+\left||S|^pe^{j\varphi(S)}-|\hat{S}|^pe^{j\varphi(\hat{S})}\right|^2
$$
